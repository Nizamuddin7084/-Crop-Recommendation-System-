#  Crop Recommendation System

##  Project Description
This is a Machine Learning project that recommends the best crop based on soil nutrients and environmental conditions like temperature, humidity, pH, and rainfall.

---

##  Objective
The goal of this project is to help farmers select the most suitable crop for better yield using data-driven predictions.

---

## Features (Input Parameters)
- Nitrogen (N)
- Phosphorus (P)
- Potassium (K)
- Temperature
- Humidity
- pH
- Rainfall

---

##  Algorithms Used
- Decision Tree
- Random Forest

---

##  Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

---

##  Project Workflow
1. Data Collection  
2. Data Preprocessing  
3. Model Training  
4. Model Evaluation  
5. Prediction  

---

## Output
The system predicts the most suitable crop based on the given input values.
